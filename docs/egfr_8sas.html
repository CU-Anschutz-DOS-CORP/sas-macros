Detailed Description

Purpose
This macro runs within a DATA step and calculates eGFR for adults using the 2009 and 2021 CKD-EPI equations
Author
Rocio Lopez
Date
07-21-2023
Version
SAS 9.4 or later
Usage
A call to this macro has all or some of these parameters:
%macro egfr(
age =
,female =
,creatinine =
,black =
,prefix =
);
Returns
The following columns will be added to the data set being created: eGFR09 eGFR21
Parameters
ageVariable containing age of the subject
femaleVariable indicating whether the candidate is female
This must be coded as 1 = female and 0 = male
creatinineVariable containing serum creatinine in mg/dL
blackVariable indicating whether the donor is of Black race
This must be coded as 1 = black and 0 = other race
prefixOptional prefix for variable names
For example, if prefix = d_ then the 2 variables created will be d_eGFR09 d_eGFR21
Note
  • This macro will only run from within a data step.
  • eGFR will only be calculated if age, female, creatinine, and race (for 20009 equation) are not missing, age is 18 or older, and creatinine level is above 0
  • CKD-EPI Creatinine Equation (2021) references:
    https://www.kidney.org/content/ckd-epi-creatinine-equation-2021
    https://www.kidney.org/professionals/kdoqi/gfr_calculator
  • CKD-EPI Creatinine Equation (2009) references:
    https://www.niddk.nih.gov/health-information/professionals/clinical-tools-patient-management/kidney-disease/laboratory-evaluation/glomerular-filtration-rate-calculators/historical
    https://www.ncbi.nlm.nih.gov/pmc/articles/PMC2763564/
Example(s)
title "eGFR";
data egfr;
input age female scr black;
%egfr(
age = age
,female = female
,creatinine = scr
,black = black
);
label egfr09 = "eGFR (2009 CKD-EPI formula)"
egfr21 = "eGFR (2021 CKD-EPI formula)";
datalines;
56 1 6.00 0
69 0 8.05 0
49 0 5.55 1
34 1 2.80 1
;
run;
proc print data=egfr; *label;
run;
title;
