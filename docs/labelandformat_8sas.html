Detailed Description

Purpose
This macro reads in a data dictionary and applies labels and formats to variables in a SAS data file
Author
Rocio Lopez
Date
11-21-2014
Version
SAS 9.4
Usage
A call to this macro has some or all of these parameters (defaults given):
%labelandformat(
data=
,newData=
,dictionary=
,guessingRows=all
,reOrder=Y
);
Returns
A SAS data set with labeled and formatted variables. The results window will show the contents of the new data set.
Note
All variables in data set should be included with a label but the format is optional
Parameters
dataSAS data file containing the variables to be labeled and formatted
newDataName of new SAS data file with labels and formats
If left blank, original data set (DATA) will be replaced
dictionaryCSV file containing the data dictionary; no quotes and no extension (e.g. ~/userid/dataDictionary)
The file must have the following columns:
Name = names of variables in the data file
Label = labels to be assigned to said variables
Format = formats to be assigned to said variables (do not include . at end)
guessingRowsSpecifies the number of rows of the dictionary file to scan to determine the appropriate length for the labels @ Default is ALL
reOrderRe-order variables based on dictionary file @ Allowed options are:
0, no, N for No, use original order
1, yes, Y for Yes, re-order based on data dictionary order
Example(s)
%labelandformat(
data = sashelp.class
,newData = class
,dictionary = test/myDataDictionary
);
