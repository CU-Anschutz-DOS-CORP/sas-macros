Detailed Description

Purpose
Checks for the number of duplicate records in a given file. Adds in potentially two new variables for total count and which one the record is.
Author
Ed Mascha, Cleveland Clinic
Maintenance
Rocio Lopez
Date
Unknown
Version
SAS 9.2 or later
Usage
A call to this macro has some or all of these parameters (defaults given):
%dup(
mainData =
,byVar =
,othervar =
,xtraSort =
,countMax = 1
,sort = 1
,mergeAll = 0
,totlName = nPerID
,ithName = nWithnID
,print = 0
);
Returns
The LISTING will show:
  • Number of duplicates on file
  • Min, max and a frequency table of observations per byVar
The macro creates the following datasets:
  • __DUPWRK - Copy of mainData excludeing all exact duplicate rows. N per Id and N within columns are added.
  • __DUPCNT - Has byVar and N per Id
  • __FIRST - Includes the first row for each byVar
Note
  • The macro checks only consecutive obs, non-consecutive duplicate obs will remain in the output data set
  • You can remove all duplicates with this macro by sorting on all variables using the XTRASORT option
Parameters
mainDataThe dataset you wish to check out
byVarThe variable(s) by which you want to check for dups, usually an ID variable
xtraSortExtra Variables to sort by
Optional, Default is None
otherVarList of other variables you want in the print out of the data
Optional, Default is None
countMaxThe number of records you expect for each variable
Will print a listing of vars not meeting this criteria
Optional, Default is 1
sortA logical for whether the data should be sorted or not
Valid options are: 1=Sort the data, 0=Data already sorted
Optional, Default is 1
mergeAllMerge the total count variable back with the original data.
Valid options are: 1=Merge, 0=Dont Merge
Optional, Default is 0
totlNameName for the new variable for total count when mergeAll=1
Optional, Default is NperID
ithNameName for the new variable for the rank count when mergeAll=1
Optional, Default is NwithnID
printAllow/Supress the proc print of the data
Valid options are: 0=Do not print, 1=Print the data
Optional, Default is 0
Example(s)
*Example 1;
%dup(
mainData = sashelp.comet
,byVar = Sample
,xtraSort = _all_
);
