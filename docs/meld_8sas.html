Detailed Description

Purpose
This macro runs within a DATA step and calculates Model For End-Stage Liver Disease (MELD), MELD-Na, and MELD 3.0 scores
Author
Rocio Lopez
Date
07-14-2023
Version
SAS 9.4 or later
Usage
A call to this macro has all or some of these parameters:
%meld(
ageAtListing =
,female =
,bilirubin =
,sodium =
,inr =
,creatinine =
,albumin =
,dialysis =
,prefix =
,meldNaAlways = 1
,cap = 0
);
Returns
The following columns will be added to the data set being created: meld, meldNa, meld3
Parameters
ageAtListingVariable containing age at time of liver transplant listing
femaleVariable indicating whether the candidate is female
This must be coded as 1 = female and 0 = male
bilirubinVariable containing bilirubin in mg/dL
sodiumVariable containing serum sodium in mEq/L
inrVariable containing INR
creatinineVariable containing serum creatinine in mg/dL
albuminVariable containing albumin in g/dL
dialysisVariable indicating whether the candidate received two or more dialysis treatments within 7 days prior to the serum creatinine test or received 24 hours of continuous veno-venous hemodialysis (CVVHD) prior to the serum creatinine test
This must be coded as 1 = yes and 0 = no
prefixOptional prefix for meld score variable names
For example, if prefix = r_ then the 3 variables created will be r_meld r_meldNa r_meld3
meldNaAlwaysCalculates MELDNa regardless of allocation policy
Allowed options are:
0 = No, only apply MELDNa if MELD is greater than 11
1 = Yes, apply MELDNa regarless of original MELD value (DEFAULT)
capRequests that the scores be capped between 6-40 as per OPTN policy
Allowed options are:
0 = No, calcuate actual score without capping (DEFAULT)
1 = Yes, score will be capped betwee 6 and 40
Note
  • This macro will only run from within a data step.
  • If dialysis is missing, calculations will assume no dialysis.
  • OPTN policy used MELD (meld) prior to 01/2016, MELD-Na (meldNa) from 01/??/2016 to 07/12/2023, and MELD 3.0 starting on 7/13/2023
  • Notice of OPTN Policy Change (MELD-Na to MELD 3.0: https://optn.transplant.hrsa.gov/media/3idbp5vq/policy-guid-change_impr-liv-alloc-meld-peld-sta-1a-sta-1b_liv.pdf
  • Reference for current policy using MELD 3.0 (as of 7/13/2023)
    https://optn.transplant.hrsa.gov/media/eavh5bf3/optn_policies.pdf (section 9.1.D)
    https://optn.transplant.hrsa.gov/media/qmsdjqst/meld-peld-calculator-user-guide.pdf
    https://optn.transplant.hrsa.gov/data/allocation-calculators/meld-calculator/
  • MELD 3.0: Kim WR, Mannalithara A, Heimbach JK, Kamath PS, Asrani SK, Biggins SW, Wood NL, Gentry SE, Kwong AJ. MELD 3.0: The Model for End-Stage Liver Disease Updated for the Modern Era. Gastroenterology. 2021 Dec;161(6):1887-1895.e4. doi: 10.1053/j.gastro.2021.08.050. Epub 2021 Sep 3. PMID: 34481845; PMCID: PMC8608337.
  • The lower limit of Serum Sodium (Na) is capped at 125, and the upper limit is capped at 137.
  • The upper limit of serum creatinine is capped at 4; in addition, if the patient had dialysis at least twice in the past week, the value for serum creatinine will be automatically adjusted to 4.0
Example(s)
*Using examples on Table 4 of Gastroenterology, 2021-12-01, Volume 161, Issue 6, Pages 1887-1895.e4;
title "MELD Scores";
data meld;
input age_at_listing female bilirubin sodium inr scr albumin dialysis;
%meld( ageAtListing = age_at_listing
,female = female
,bilirubin = bilirubin
,sodium = sodium
,inr = inr
,creatinine = scr
,albumin = albumin
,dialysis = dialysis
,prefix = r_);
label r_meld = "Recipient MELD at time of transplant"
r_meldNa = "Recipient MELDNa at time of transplant"
r_meld3 = "Recipient MELD 3.0 at time of transplant";
datalines;
20 0 2.5 131 1.0 1.2 3.8 0
20 1 2.5 131 1.0 1.2 3.8 0
20 0 6.0 131 1.5 1.5 3.5 0
20 0 6.0 131 1.5 1.5 2.2 0
20 1 6.0 131 1.5 1.5 2.2 0
20 0 12.0 128 2.2 1.8 2.0 0
20 0 12.0 128 2.2 2.8 2.0 0
20 1 12.0 128 2.2 2.8 2.0 0
20 1 12.0 120 0.9 4.5 3.5 0
10 1 12.0 120 0.9 4.5 3.5 0
20 0 12.0 120 0.9 4.5 3.5 0
10 0 12.0 120 0.9 4.5 3.5 0
;
run;
proc print data=meld; *label;
run; title;
