Detailed Description

Purpose
This macro generates a report of missing data, overall or by group. It can handle numeric and character data and identifies all valid SAS missing values.
Author
Rocio Lopez
Date
12-06-2018
Version
SAS 9.2 or later
Usage
A call to this macro has some or all of these parameters (defaults given):
%missing(
data =
,byVar =
,vList = _all_
,highLightVal = 30
,missPattern = 0
,printAll = 0
,sortBy = descending _nmiss
,reverseCol = 0
,out = _missing_
,rtffile =
,page = portrait
,xlsxfile =
,debug = 0);
Returns
A table displaying % of all observations that are missing each particular variable
Parameters
dataThe dataset you wish to check out
byVarOptional group variable by which to report missing values
listList of variables to check for missing values
Default is ALL
missPatternAllows to request a description of missing pattern
Allowed options are:
0 = No, do not incude (DEFAULT)
1 = Yes, include
highLightValThis is a single numeric value (0-100) that specifies which % missing values to highlight; anything with % missing greater than or equal to HIGHLIGHTVAL will appear in red bolded font
Default is 30
printAllRequest a report including all variables in LIST
Allowed options are:
0 = No, only include those with >0% missing (DEFAULT)
1 = Yes, include all variables
sortByAllows sorting of OUT data set generated for report
Default is DESCENDING _NMISS
reverseColReverse order of display for group variable (byVar)
Allowed options are:
0 = No, display in ascending order (DEFAULT)
1 = Yes, reverse and display in descending order
outName of output dataset
Default is missing
rtfFileLocation and name for output RTF file sourrounded by double quotes (e.g. "~userid/consult/missReport.rtf")
pagePage orientation of RTF file
Allowed options are:
PORTRAIT (DEFAULT)
LANDSCAPE
xlsxFileLocation and name for output RTF file sourrounded by double quotes (e.g. "~userid/consult/missReport.xlsx")
debugDebugging option
Allowed options are:
0 = Off (DEFAULT)
1 = On
Example(s)
data cars;
set sashelp.cars;
if ranuni(987) le .5 then call missing(msrp);
if ranuni(987) le .2 then call missing(mpg_city, mpg_highway);
if ranuni(987) le .15 then call missing(drivetrain);
if ranuni(987) le .05 then call missing(origin, weight);
label DriveTrain = "Drive Train"
MPG_City = "City MPG"
MPG_Highway = "Highway MPG";
run;
%missing(data=cars);
