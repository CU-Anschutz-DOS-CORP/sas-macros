Detailed Description

Purpose
This macro will import all *.XPT files found in the specified directory
Author
Rocio Lopez
Date
09-19-2018
Version
SAS 9.4 or later
Usage
A call to this macro has some or all of these parameters (defaults given):
%imptallxpt(
directory=
,library=work
);
Returns
One SAS data set per each imported XPT file
Note
Subfolders of directory will not be searched
Parameters
directoryFolder where all .XPT files are found; do not use quotes
librarySAS library in which to store imported files
Default is WORK
See also
getnhanesdata.sas, nhanesfmtscrape.sas
Example(s)
%imptallxpt(directory=./nhanes2015to16);
Revision History
03-31-2023 Change to allow use of backward or forward slashes in directory name
12-14-2022 Updated to run in PC SAS as well as Linux SAS
03-21-2019 Changed how upper/lower case files are searched for in Linux and added check for when folder has no .XPT files
