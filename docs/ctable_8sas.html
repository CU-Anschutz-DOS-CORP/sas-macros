Detailed Description

Purpose
This macro generates a table assessing associationg between 1 continuous outcome and categorical or continuous variables.
Author
Rocio Lopez
Date
06-10-2016
Version
SAS 9.4 or later
Usage
A call to this macro has some or all of these parameters (defaults given):
%ctable(
ds=
,outcome=
,catList=
,stat=mean
,contList=
,corrType=P
,subset=
,outSs=_outds
,nFmt= __countf.
,pFmt= pvaluef.
,estFmt= __estf.
,rhoFmt=8.2
,tblTitle=
,rtfFile=
,cWidth1=250
,cWidth2=50
,cWidth3=225
,nTables=1
,deBug=0
);
Returns
One or 2 report-ready tables and a SAS data set with all results.
Parameters
dsSAS data set name
outcomeName of continuous outcome variable (only 1 allowed)
catlistList of categorical variable(s) to analyze
contlistList of continuous variable(s) to analyze
subsetExpression to subset population. This will be used in a WHERE statement.
statOutcome summary statistics presented for categorical variables.
Options are:
MEAN for mean +/- sd and ANOVA (DEFAULT)
MEDIAN for median [P25, P75] with Kruska-Wallis test
corrtypeCorrelation type for association between outcome and continuous variables
Options are:
P for Pearson's correlation (95% CI) (DEFAULT)
S for Spearman's correlation (95% CI)
outdsSAS data set that will contain final output
nfmtFormat for N column
Default is __countf. (defined within macro)
pfmtFormat for p-value
Default is pvaluef. (defined within macro)
estfmtFormat for mean, sd, percentiles
Default is __esttf. (defined within macro)
rhofmtFormat for correlations
Default is 8.2
tbltitleTile for table surrounded by ""
Default is "Univariate relationships between &outcome and variables of interest"
rtffileRTF output location and file name; surrounded by " " (e.g. "./Example.rtf")
cwidth1Width for label column in .rtf file
Default is 250
cwidth2Width for N column in .rtf file
Default is 50
cwidth3Width for statistics column in .rtf file
Default is 225
ntablesNumber of output tables; this will only take effect if BOTH catlist and contlist are provided
Options are:
1 for 1 output table with catlist and contlist (DEFAULT)
2 for 2 output tables, 1 with catlist and another with contlist
debugTurn on/off mlogic and mlogicnest for error checking
Options are:
0 for off (DEFAULT)
1 for on
Note
  • Variables should be labeled and formatted.
  • Note that if you are runing in a windowing environment the plus/minus sign when reporting means and sds will appear as (ESC){unicode '00B1'x}. This will be properly displayed in RTF.
  • This macro uses the AUTOCALL CMPRES macro. Make sure this is accessible via sasautos = (sasautos) or, if not, loaded into your session
  • Datasets created and deleted during run: null _med _p _temp _rho _rho2 _rho3 _lbs _varlabels _misout tmiss _subset
Example(s)
%ctable(
ds=sashelp.cars
,outcome=MSRP
,catList=Type DriveTrain
,stat=median
,contList=MPG_City MPG_Highway
,cwidth3=350
,rtfFile="./CTABLE.rtf"
);
