Detailed Description

Purpose
This program retrieves all the .LOG files in the specified directory and scans them for errors, warnings and notes of concern.
Author
Richann Watson, Experis
Updates/Maintenance
Rocio Lopez
Date
2017
Version
SAS 9.4
Usage
A call to this macro has some or all of these parameters (defaults given):
%logCheck(
loc =
,loc2 =
,fnm =
,delm = @
,out =
);
Returns
An RTF file with an entry for each .LOG file specifying whether the file is clean or listing any errors, warning or notes of concern
Note
Please see the following SAS Global Forum and MSUG papers for details:
This macro: http://support.sas.com/resources/papers/proceedings17/1173-2017.pdf
Updated version from author: https://www.mwsug.org/proceedings/2017/TT/MWSUG-2017-TT02.pdf
Parameters
locDirectory where the log files can be found, do no use quotes
loc2Directory where report is to be stored, do no use quotes
Default is same directory where log files are (LOC)
fnmAllows to speify which types of files to look at
e.g., Only .log files starting with t_ (tables); separate types of files by delimiter indicated in the delm macro parameter (e.g., t_@f_)
By default it will look at all .log files
delmDelimiter used to separate types of files
Default is @
outOptional name for output report
If left blank, ALL_CHECKLOGS is used
Example(s)
%logCheck(loc = ./logFiles);
