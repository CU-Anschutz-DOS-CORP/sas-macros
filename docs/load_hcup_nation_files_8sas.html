Detailed Description

Purpose
This macro will download any available SAS load files from the HCUP website and proceed to execute these. The macro can also be used to run any manually downloaded SAS load programs.
Author
Rocio Lopez
Date
09-22-2023
Version
SAS 9.4 or later
Usage
A call to this macro has some or all of these parameters (defaults given):
%loadHcupNationFiles(
nwDatabase =
,year =
,baseDir =
,overwrite = 1
,getSasFiles = 1
,runSas = 1
,debug = 0
);
Returns
.SAS and .SAS7BDAT files for the specified database/year will be stored in the <BASEDIR>/<NWDATABASE>/<YEAR> folder.
Parameters
nwDatabaseSpecify the HCUP nationwide database to process
Allowed options are: NASS, NEDS, NIS, NRD, or KID
baseDirSpecify the base directory where required materials are saved and where the output will be stored
Do not include the database name and year in this path
yearYear of data that will be processed (yyyy)
overwriteLogical stating whether to overwrite existing files
Allowed options are:
0 for no
1 for yes (DEFAULT)
getSasFilesLogical stating whether to download all available SAS load programs
Allowed options are:
0 for no
1 for yes (DEFAULT)
runSasLogical stating whether to run SAS load programs and save permanent copies of the data files as .SAS7BDAT
Allowed options are:
0 for no
1 for yes (DEFAULT)
Note that this will overwrite any existing files in <BASEDIR>/<NWDATABASE>/<YEAR> regardless of whether you set overwrite to 0
debugTurns on/off several internal logging techniques
Allowed options are:
0 for off (DEFAULT)
1 for on
Note
  • This macro has been tested with NRD data on a Windows environment only.
  • If requesting to run the SAS load programs, user must have purchased, extracted, and saved the HCUP files in <BASEDIR>/<NWDATABASE>/<YEAR>. Do not rename any file(s).
  • .SAS and .SAS7BDAT files files will be saved in the <BASEDIR>/<NWDATABASE>/<YEAR> folder.
  • Go to https://hcup-us.ahrq.gov/ to purchase data and/or download supporting materials


Example(s)
%processHcupNationFiles(
nwDatabase = NRD
,year = 2020
,baseDir = ./hcup
);
